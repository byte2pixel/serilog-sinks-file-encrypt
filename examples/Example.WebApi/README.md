# Example.WebApi

This example demonstrates how to set up Serilog logging in an ASP.NET Core Web API project.

## Setup Instructions

1. Create a new ASP.NET Core Web API project if you don't have one already:
```bash
dotnet new webapi -n Example.WebApi
cd Example.WebApi
```

2. Add the necessary Serilog packages:
```bash
dotnet add package Serilog.AspNetCore
dotnet add package Serilog.Sinks.File
```
